# Criando-um-Relat-rio-Vendas-e-Lucros-com-Data-Analytics-com-Power-BI

# O que é:

Este projeto tem como objetivo a criação de um dashboard interativo no Power BI para análise de vendas e lucratividade, permitindo uma visão estratégica dos dados e suporte à tomada de decisão. Através da aplicação de conceitos de Data Analytics, o relatório fornece insights sobre desempenho comercial, comportamento de vendas e indicadores financeiros.

---

# Ferramenta Utilizada:
Power BI Desktop

---

# Como Utilizar:

1. Faça o download ou clone este repositório
2. Abra o arquivo .pbix no Power BI Desktop
3. Atualize a base de dados, se necessário
4. Explore os dashboards interativos

# Pré-requisitos:
Power BI Desktop instalado
